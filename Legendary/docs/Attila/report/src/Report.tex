\title{Self-driven Miniature Vehicle for Carolo Cup}
\author{Attila Nagy}
\department{Department of Computer Science and Engineering}
\thesisin{Computer Science}
\copyrightyear{November, 2013}

\firstabstract{
  Carolo Cup is an international student competition for self-driven miniature
  vehicles organized annually in Germany. Such vehicles above providing certain
  functionalities must fulfill extra requirements, such as: time-constraints,
  and performance.  The basic functionalities required for the Carolo Cup
  competition are lane following, crash avoidance, overtaking, intersection handling and parking. This
  report provides solutions for one of these functionalities, both
  describing its theoritical background and presenting a working
  implementation. The functionality covered by this report is
  lane following.
}
